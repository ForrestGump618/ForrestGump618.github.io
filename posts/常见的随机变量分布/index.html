离散型随机变量的分布

二项分布

定义

二项分布是一种离散概率分布，常用于描述在固定次数的独立实验中，成功次数的概率分布。假设每次实验只有两种可能的结果：成功或失败，且每次实验成功的概率是相同的。如果进行

nnn

次独立的伯努利试验，每次试验成功的概率为

ppp

，那么在这些试验中成功

kkk

次的概率可以用二项分布来描述。

公式表示

二项分布的概率质量函数（PMF）为：

P(X=k)=(nk)pk(1−p)n−kP(X = k) = \binom{n}{k} p^k (1-p)^{n-k}P(X=k)=(kn​)pk(1−p)n−k


其中：

  • nnn

    是试验次数。

  • kkk

    是成功次数。

  • ppp

    是每次试验成功的概率。

推导

二项分布的推导可以从伯努利试验的性质开始。假设每次试验成功的概率为

泊松分布

定义

泊松分布是一种离散概率分布，常用于描述在固定时间或空间内，某事件随机发生的次数的概率分布。假设事件的发生是相互独立的，并且每个时间或空间单位内事件发生的概率是相同的。如果在一个时间或空间单位内事件的发生次数为

kkk

，那么这个概率可以用泊松分布来描述。

公式表示

泊松分布的概率质量函数（PMF）为：

P(X=k)=e−λλkk!P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}P(X=k)=k!e−λλk​


其中：

  • λ\lambdaλ

    是单位时间或空间内事件发生的平均次数。

  • kkk

    是事件的发生次数。

  • eee

    是自然对数的底数，约等于 2.71828。

[0,1)[0,1)[0,1)

泊松分布在许多领域都有应用，包括通信、交通流量分析、保险业等。通过泊松分布，我们可以预测在一定条件下事件发生的次数，从而做出更合理的决策。

  • 在保险业中，泊松分布可以用来预测在一定时间内发生的索赔次数。

超几何分布

定义

超几何分布是一种离散概率分布，常用于描述从有限总体中抽取样本时，特定类型元素出现的次数的概率分布。假设总体中有

NNN

个元素，其中

KKK

个是成功元素（即我们感兴趣的类型）。如果我们从这个总体中随机抽取

nnn

个元素，那么抽取到

kkk

个成功元素的概率可以用超几何分布来描述。

公式表示

超几何分布的概率质量函数（PMF）为：

P(X=k)=(Kk)(N−Kn−k)(Nn)P(X = k) = \frac{\binom{K}{k} \binom{N-K}{n-k}}{\binom{N}{n}}P(X=k)=(nN​)(kK​)(n−kN−K​)​

其中：

  • NNN

    是总体元素的总数。

  • KKK

    是总体中成功元素的数量。

  • nnn

    是抽取的样本大小。

  • kkk

    是抽取到的成功元素的数量。

  • (ab)\binom{a}{b}(ba​)

    表示从

    aaa

    bbb

超几何分布在许多领域都有应用，包括统计学、生物统计学、质量控制等。通过超几何分布，我们可以预测在特定条件下抽取到特定类型元素的概率，从而做出更合理的决策。需要注意的是，如果

nN\frac {n}{N}Nn​

例如：

  • 在质量控制中，超几何分布可以用来预测从一批产品中抽取的样本中不合格产品的数量。
  • 在生物统计学中，超几何分布可以用来预测从某个种群中抽取的样本中具有某种特征的个体数量。
  • 在市场研究中，超几何分布可以用来预测从消费者群体中抽取的样本中具有某种偏好的消费者数量。

负二项分布

定义

负二项分布（也称为伽玛-泊松分布或Pascal分布）是一种离散概率分布，常用于描述在固定数量的事件中，某个特定事件首次发生之前其他事件随机发生的次数的概率分布。与泊松分布类似，负二项分布在描述事件的随机发生次数方面也有广泛的应用，但它更侧重于在达到某个特定事件之前发生的事件次数。

公式表示

负二项分布的概率质量函数（PMF）为：

P(X=k)=(k−1r−1)pr(1−p)k−rP(X = k) = \binom{k-1}{r-1} p^r (1-p)^{k-r}P(X=k)=(r−1k−1​)pr(1−p)k−r


其中：

  • rrr

    是达到特定事件所需的次数。

  • ppp

    是每次试验中特定事件发生的概率。

  • kkk

    是在首次达到特定事件之前发生的事件总数。

  • (nk)\binom{n}{k}(kn​)

    是组合数，表示从

    nnn

    个不同元素中取出

    kkk

    个元素的组合方式数。

推导

假设我们有一个试验序列，每次试验中特定事件的发生概率为

ppp

，不发生的概率为

1−p1-p1−p

。我们感兴趣的是在首次达到

rrr

k−rk-rk−r

。

根据二项分布的性质，我们知道在

k−1k-1k−1

次试验中恰好有

P(X=k−1)=(k−1r−1)pr−1(1−p)k−rP(X = k-1) = \binom{k-1}{r-1} p^{r-1} (1-p)^{k-r}P(X=k−1)=(r−1k−1​)pr−1(1−p)k−r

由于在第

kkk

ppp

来得到负二项分布的 PMF：

P(X=k)=(k−1r−1)pr(1−p)k−rP(X = k) = \binom{k-1}{r-1} p^r (1-p)^{k-r}P(X=k)=(r−1k−1​)pr(1−p)k−r

负二项分布的性质

期望值

负二项分布的期望值（均值）是：

E(X)=r(1−p)pE(X) = \frac{r(1-p)}{p}E(X)=pr(1−p)​

方差

负二项分布的方差是：

Var(X)=r(1−p)p2\text{Var}(X) = \frac{r(1-p)}{p^2}Var(X)=p2r(1−p)​

应用

负二项分布在许多领域都有应用，尤其是在需要预测在达到某个特定事件之前发生的事件次数的场景中。以下是一些具体应用实例：

  • 体育比赛：在篮球比赛中，负二项分布可以用来预测在首次投中三分球之前，球员投篮不中的次数。
  • 医学研究：在药物试验中，负二项分布可以用来预测在首次观察到治疗效果之前，需要进行多少次试验。
  • 工业生产：在产品质量控制中，负二项分布可以用来预测在首次生产出合格产品之前，需要进行多少次生产尝试。

通过这些应用，负二项分布帮助我们更好地理解和预测在特定条件下事件的发生次数，从而做出更合理的决策和规划。
负二项分布是二项分布的“反其道而行之”，二项分布是定下总抽样个数

nnn

，把事件个数

XXX

作为变量；儿负二项分布则恰恰相反，它顶下事件个数

rrr

，把总抽样次数减去

rrr

作为变量。

r=1r=1r=1

时，可以得到几何分布：

P(X=k)=p(1−p)k−1P(X = k) = p (1-p)^{k-1}P(X=k)=p(1−p)k−1

正态分布

定义

正态分布，也称为高斯分布，是一种在自然和社会科学领域中极为常见的连续概率分布。它描述了大量独立随机变量的平均值接近其总体均值的概率分布。正态分布的图形呈钟形曲线，左右对称，中间高，两端低。

公式表示

正态分布的概率密度函数（PDF）为：

f(x)=12πσ2e−(x−μ)22σ2f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}f(x)=2πσ2​1​e−2σ2(x−μ)2​

其中：

  • mumumu

    是分布的均值。

  • sigmasigmasigma

    是分布的标准差。

  • sigma2sigma^2sigma2

    是方差。

推导

正态分布的推导可以从中心极限定理开始，该定理指出，大量独立随机变量的平均值，无论这些变量本身是什么分布，当样本量足够大时，其分布都会接近正态分布。正态分布的数学形式可以通过以下方式推导：

  1. 假设随机变量

    XXX

    服从正态分布

    N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2)

    。
  2. 正态分布的概率密度函数可以通过最大化熵的方式得到，即在给定均值和方差的情况下，使得分布的熵最大。

正态分布的性质

期望值

正态分布的期望值（均值）是：

E(X)=μE(X) = \muE(X)=μ

方差

正态分布的方差是：

Var(X)=σ2\text{Var}(X) = \sigma^2Var(X)=σ2

标准差

正态分布的标准差是方差的平方根：

σ=Var(X)\sigma = \sqrt{\text{Var}(X)}σ=Var(X)​

正态分布的一个重要性质是其数据分布的集中趋势。根据正态分布的68-95-99.7规则：

  • 大约68%的数据值落在

    mu±σmu \pm \sigmamu±σ

    范围内。
  • 大约95%的数据值落在

    mu±2σmu \pm 2\sigmamu±2σ

    范围内。
  • 大约99.7%的数据值落在

    mu±3σmu \pm 3\sigmamu±3σ

    范围内。

应用

正态分布在许多领域都有应用，包括统计学、物理学、经济学、工程学等。通过正态分布，我们可以预测在特定条件下数据的分布情况，从而做出更合理的决策。以下是一些具体应用示例：

  • 统计学：在统计学中，正态分布被用来估计总体参数，如均值和方差。
  • 物理学：在物理学中，正态分布可以用来描述粒子的位置和速度分布。
  • 经济学：在经济学中，正态分布可以用来预测股票价格的变化。
  • 工程学：在工程学中，正态分布可以用来预测机械部件的尺寸分布。

正态分布的广泛应用使其成为统计学和数据分析中不可或缺的工具。

指数分布

定义

指数分布是一种连续概率分布，常用于描述独立随机事件发生的时间间隔。这种分布广泛应用于描述设备故障时间、电话呼叫到达时间、放射性原子的衰变时间等。指数分布的图形呈单峰曲线，随着值的增加而逐渐减小。

生物数据中数据分布的应用

二项分布 (Binomial Distribution)

  • 描述：用于描述在固定次数的独立实验中，成功次数的概率分布。
  • 公式：

    P(X=k)=(nk)pk(1−p)n−kP(X = k) = \binom{n}{k} p^k (1-p)^{n-k}P(X=k)=(kn​)pk(1−p)n−k

  • 应用示例：
    • 基因测序中，某个特定突变在样本中的出现次数。
    • 药物筛选中，药物对细胞的杀伤效果的实验结果。

泊松分布 (Poisson Distribution)

  • 描述：用于描述在固定时间或空间内，某事件发生次数的概率分布。
  • 公式：

    P(X=k)=e−λλkk!P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}P(X=k)=k!e−λλk​

  • 应用示例：
    • 基因组中某基因的突变频率。
    • 蛋白质组学中，特定蛋白质的表达量。

超几何分布 (Hypergeometric Distribution)

  • 描述：用于描述从有限总体中抽取样本时，成功次数的概率分布。
  • 公式：

    P(X=k)=(Kk)(N−Kn−k)(Nn)P(X = k) = \frac{\binom{K}{k} \binom{N-K}{n-k}}{\binom{N}{n}}P(X=k)=(nN​)(kK​)(n−kN−K​)​

  • 应用示例：
    • 从特定群体中抽取样本，计算样本中某种基因型的比例。
    • 从特定组织中抽取细胞，分析细胞类型的比例。

负二项分布 (Negative Binomial Distribution)

  • 描述：用于描述在一系列独立实验中，达到固定成功次数所需的实验次数的概率分布。
  • 公式：

    P(X=k)=(k−1r−1)pr(1−p)k−rP(X = k) = \binom{k-1}{r-1} p^r (1-p)^{k-r}P(X=k)=(r−1k−1​)pr(1−p)k−r

  • 应用示例：
    • 基因测序中，达到一定覆盖度所需的测序深度。
    • 药物筛选中，达到一定效果所需的药物浓度。

正态分布 (Normal Distribution)

  • 描述：用于描述连续变量的概率分布，假设数据呈对称分布。
  • 公式：

    P(x)=12πσ2e−(x−μ)22σ2P(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}P(x)=2πσ2​1​e−2σ2(x−μ)2​

  • 应用示例：
    • 蛋白质表达量的分布。
    • 基因表达量的标准化值。

指数分布 (Exponential Distribution)

  • 描述：用于描述连续变量的概率分布，假设事件发生的时间间隔是独立且同分布的。
  • 公式：

    P(x)=λe−λxP(x) = \lambda e^{-\lambda x}P(x)=λe−λx

  • 描述：用于描述连续变量的概率分布，常用于描述寿命数据。

    P(x)=kλ(xλ)k−1e−(xλ)kP(x) = \frac{k}{\lambda} (\frac{x}{\lambda})^{k-1} e^{-(\frac{x}{\lambda})^k}P(x)=λk​(λx​)k−1e−(λx​)k
