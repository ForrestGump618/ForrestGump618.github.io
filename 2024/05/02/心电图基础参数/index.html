心电图基本参数和解释

▲     导联示意图

 

P波：

n  I、II、aVF、V4~V6导联为正（心房除极从右上到左）；

n  P波时间 < 0.12s；

n  P波振幅：肢体导联 < 0.25mv（2.5小格）；胸导联 < 0.20mv（2小格）

n  PR间期 0.12~0.20s，<0.20s；

 

QRS波群：

n  QRS波群时间 < 0.11s，0.06~0.10s；

n  胸导联：

V1、V2：rS型（r为室间隔左向右除极，S是心室除极，左心室从室间隔向心尖除极占上风，故V1呈S波）；

V1导联的r波振幅小于1.0mV。

V5、V6：qR、qRs、Rs或R型（R是主要的，和V1、V2的S波是对偶的；q代表着室间隔从左向右除极，s或为右心室除极）【R是主要的，q次之，s更次之】；

R和S的移行：

判定转位的依据【V3、V4正对室间隔】，V2 S最深、V5 R最高【倒数第二】；

V1 R/S < 1；V5 R/S > 1（这两个导联是判定束支传导阻滞的重要依据）；

n  肢体导联：

I、II主波向上（与除极时左心室除极向量一致，而前者是除极的主体）；I R < 1.5mv；

III主波不定【利用上面左图可计算代数和，并判定心电向量在不同象限（或区域）】；

aVR 主波向下（背离I、II导联），呈QS、rS、rSr`或Qr型（此导联观察右心室流出道，后者即为心脏最后除极的地方，故可见终末的r波；在r波前可记录到左心室背对此导联的除极波，形成S或者Q；室间隔从左向右除极可负可正）；R< 0.5mv；

aVL和aVF主波向上，呈qR、Rs或R型（正向对左心室除极，q波或室间隔除极波，s波可为右心室除极波）；aVL R < 1.2mv；aVF R< 2.0mv；

 

Q波：

n  除了aVR和III导联，不超过0.03s；

n  III导联 < 0.04s；

n  aVR可见较宽的Q或者QS；

n  Q < 1/4 R；

 

ST段：

n  下移不超过0.05mv（半个小格）；

n  V2和V3可上抬0.2mv以上；

n  V4~V6不会高于0.1mv；

n  V2~V5、II、III和aVF可见J点上移，ST段凹面上抬，为早期复极（心外膜先复极，向量指向这些导联）；

T波：

n  方向和QRS主波方向一致，否则可考虑QRS变异，见于室性早搏、WPW或束支传导阻滞；

n  若V1向上，则一般V2~V6向上；

n  除了III、aVL和V1~V3，其他导联均应该不低于1/10 R，胸导联可高达1.2~1.5mv。

 

PJ间期：

n  PJ间期指P波起点到QRS末端J点的时长；

n  预激综合征PJ间期正常（＜0.27s）；

n  室内传导阻滞、心室异位起搏等PJ间期延长。

 

QT间期：

n  不计算U波，QRS起点到T波终点的间距（一般认为U波是浦肯野纤维的复极，严格来说不属于心室电活动）；

n  QT间期的正常范围为0.32s-0.44s，男性QT间期>0.45s和女性QT间期>0.46s则为QT间期延长；

n        。

 

U波：

n  浦肯野纤维的复极波；

n  低钾血症时，由于浦肯野纤维的钾离子通道对钾离子浓度更加敏感，因此复极化更慢，所以会出现特征性的U波。
